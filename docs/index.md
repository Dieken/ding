# 顶功 · 集萃

2005 年，戴石麟先生的顶功输入方案开山之作「声笔码」及「声笔飞码」横空出世，输入法界的新时代悄然来临。越来越多的人从质疑到了解再到恍然大悟、为己所用，在二十年间创作出了各式各样的顶功输入方案。

何为顶功？有哪些成熟的作品？如何制作新的顶功输入方案？这些是本书回答的。愿您也能感受到顶功的趣味。

析顶功之理，赏万码之韵。
